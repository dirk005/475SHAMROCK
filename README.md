# Sass-starter-template
- This is just starter files for a sass web page

## To run this project

- clone this here repository
- run these following commands on your terminal/ cmd prompt
  - cd Sass-starter-template
  - npm install
  - npm start

## To build this project
- run these following commands on your terminal/ cmd prompt
  - cd Sass-starter-template
  - npm run build:css
